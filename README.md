## 六爻排盘工具
此乃**本人自用六爻排盘工具。**<br/>
支持汉字起卦、数字起卦、硬币或者太极丸起卦等主流起卦方式。<br/>
基于Expo开发，组件库使用tamagui，lunisolar实现四柱八字的解析和转换，cnchar用于汉字的拆字。适配安卓、iOS、Web。<br/>
